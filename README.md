# Weather App

This is a weather app which serves as a practice for using Promises, asynchronous callback functions, and working with APIs.

## Features

- Loads device location weather data when permitted
- Search weather data from a location
- Displays data from current date to next week
- Click through the weeks weather data forecast to see a more detailed result

This project was create as a part of the Odin Project:
https://www.theodinproject.com/lessons/javascript-weather-app
